# Task Tracker 📝

A simple task tracking app built with **React**, **Redux**, and **Bootstrap**.

## 🚀 Features

- Add tasks ✅
- Mark tasks as completed 🟢
- Delete tasks ❌
- Filter tasks by:
  - All 🔍
  - Completed ✅
  - Pending ⏳

## 🛠️ Tech Stack

- React
- Redux (with Redux Persist)
- Bootstrap
- React-Bootstrap
